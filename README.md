# NivoRd Figure Scripts

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19430235.svg)](https://doi.org/10.5281/zenodo.19430235)

Scripts to reproduce figures from the NivoRd manuscript (Letter to Blood).

**Data deposit:** https://doi.org/10.5281/zenodo.19430235 (concept DOI, always resolves to the latest version). Current version: [10.5281/zenodo.19711926](https://doi.org/10.5281/zenodo.19711926). Data are under embargo until 2028-01-01.

## Layout

Scripts are numbered by manuscript order. Supplementary Figure S1 (clinical outcomes — swimmer, Kaplan-Meier curves, CONSORT) is generated by the clinical team and is not included here.

```
2-Scripts/
├── README.md
├── 01a-IMC_analysis.ipynb                     # IMC preprocessing, clustering, CN, S2C dotplots
├── 01b-IMC_figures.Rmd                        # IMC downstream plots (Fig 1B/1C/1D/1F/2D, S2D, S3)
├── 02-Figure1E_IMC_composites.py              # 4-channel TIFF composites
├── 03-Figure2A_immune_umap.py                 # CD138⁻ immune UMAP
├── 04-Figure2B_macrophage_volcano.py          # Post-treatment macrophage DGE
├── 05-Figure2C_macrophage_gsea.py             # Hallmark GSEA
├── 06-Figure2EF_external_validation.py        # Zavidij external cohort
├── 07-FigureS2A_marker_heatmap.py             # Immune cell-type markers
├── 08-FigureS2B_macrophage_umap.py            # Macrophage-highlighted UMAP
├── 09-FigureS2E_tumor_umap.py                 # CD138⁺ tumor UMAP by patient
├── 10-FigureS2F_cytogenetic_heatmap.py        # Cytogenetic validation
├── 11-FigureS4A_pretreatment_volcano.py       # Pre-treatment macrophage volcano
└── 12-FigureS4BC_cell_interactions.py         # LIANA ligand-receptor axes
```

## Pipeline overview

```
IMC raw 34-channel TIFFs ─► (image import) ─► IMC-import.h5ad
IMC-import.h5ad ─► 01a-IMC_analysis.ipynb ─► phenotyped-obs.csv + CN CSVs + S2C dotplots
phenotyped-obs.csv + clinical CSVs ─► 01b-IMC_figures.Rmd ─► Fig 1B/1C/1D/1F/2D + Supp boxplots

scRNA-seq h5ad ─► 03–05, 07–12 ─► Fig 2A/2B/2C + Supp figures
Zavidij GEO (auto-downloaded) ─► 06-Figure2EF_external_validation.py ─► Fig 2E/2F
```

## Configuring data paths

The IMC notebook and R markdown expect processed data under a single root directory, `DATA_DIR`, with the layout:

```
$DATA_DIR/
├── nivo_clinical.csv
├── nivo_md.csv
├── nivo_MSPike.csv
├── outs/
│   ├── phenotyped-obs.csv
│   ├── combat_all_markers.csv
│   └── import.h5ad
├── CN_data/
│   ├── pheno_8clus-cells_data.csv
│   └── pheno_8clus-sample_pct.csv
└── figures/
```

Set the root via the environment variable `NIVO_IMC_DATA_DIR`:

```bash
export NIVO_IMC_DATA_DIR=/path/to/nivo_imc_data/
```

Both `01b-IMC_figures.Rmd` (R) and `01a-IMC_analysis.ipynb` (Python) read this variable and fall back to `/mnt/disks/data/imc/Nivo/` (the original processing environment) if unset.

## Usage

```bash
# scRNA-seq figures (need h5ad files)
python 03-Figure2A_immune_umap.py

# IMC figures (after running 01a notebook to generate CSVs)
export NIVO_IMC_DATA_DIR=/path/to/nivo_imc_data/
Rscript -e "rmarkdown::render('01b-IMC_figures.Rmd')"

# IMC composites (need raw 34-channel TIFFs)
python 02-Figure1E_IMC_composites.py --input path/to/tiffs --output path/to/out

# External cohort (auto-downloads from GEO)
python 06-Figure2EF_external_validation.py
```

Each script outputs PDF, PNG, and SVG at 300 DPI unless otherwise noted.
